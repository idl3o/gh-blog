StreamAccessContract

Solidity 0.8.17 Audited ERC-1155 Compatible Upgradeable

The StreamAccessContract manages content access permissions on the Web3 Crypto Streaming platform. It maintains a secure registry of which users have access to what content, for how long, and under what conditions.

This contract serves as the gatekeeper for all platform content, providing verifiable proof of access rights that can be used by the front-end application and decentralized storage systems to permit content delivery.

Key Features

  • Granular access control for digital content
  • Time-based access expiration
  • Creator-controlled access management
  • Subscription and one-time access support
  • Integration with payment systems
  • On-chain verification of viewing rights

Access Management

grantAccess(address viewer, bytes32 contentId, uint256 expiration) â†’ bool

Grants access to specific content for a viewer until the given expiration timestamp. Can only be called by the content creator or an authorized payment contract.

Parameters

Name Type Description
viewer address The address of the user being granted access
contentId bytes32 The unique identifier of the content
expiration uint256 Timestamp when the access will expire (in seconds since epoch)

Returns

bool: True if access was successfully granted

Example

// Grant access to content for 30 days
function purchaseContent(bytes32 contentId) public payable {
    // Verify payment
    require(msg.value >= contentPrice, "Insufficient payment");

    // Transfer payment to creator
    payable(contentCreator).transfer(msg.value);

    // Grant access
    uint256 thirtyDaysFromNow = block.timestamp + 30 days;
    streamAccessContract.grantAccess(msg.sender, contentId, thirtyDaysFromNow);
}

Events

The contract emits the following events to track access changes:

  • AccessGranted(address indexed viewer, bytes32 indexed contentId, uint256 expiration)
  • AccessRevoked(address indexed viewer, bytes32 indexed contentId)
  • AccessExtended(address indexed viewer, bytes32 indexed contentId, uint256 newExpiration)
