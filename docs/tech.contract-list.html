This catalog provides an overview of all smart contracts used in the Web3 Crypto Streaming Service platform. Each contract serves a specific purpose in the ecosystem, from content access control to payment processing and governance.

Note: All contracts are audited by third-party security firms and follow industry best practices for security and gas optimization.

Core Contracts

These contracts form the foundation of the platform's functionality.

StreamAccessContract

Audited ERC-1155

Manages content access permissions and viewer access control. This contract verifies whether a user has the right to access specific content based on their purchases or subscriptions.

Version: 1.2.0
View Documentation

ProofOfExistenceContract

Audited

Records content ownership and creation timestamps on the blockchain, providing verifiable proof of content existence and authorship.

Version: 1.0.3
View Documentation

ContentRegistry

Audited

Central registry for all platform content metadata, including creator information, content descriptions, and IPFS content hashes.

Version: 1.1.2
View Documentation

Access Control

These contracts manage permissions and access rights throughout the platform.

CreatorRegistry

Audited

Manages creator profiles, verification status, and platform permissions.

RoleManager

Audited

Manages role-based access control for platform administrators and moderators.

Financial

These contracts handle payments, tokens, and economic functionality.

StreamPaymentContract

Audited

Processes payments between viewers and creators, supporting one-time payments, subscriptions, and streaming payments.

Version: 1.3.1
View Documentation

StreamToken

Audited ERC-20

The native utility token for the platform, used for payments, rewards, and governance.

StakingContract

Audited

Enables token staking for creators and viewers, providing rewards and platform benefits.

Governance

These contracts facilitate decentralized decision-making and platform governance.

GovernanceContract

Audited

Manages proposal creation, voting, and execution for platform governance.

Timelock

Audited

Enforces time delays for governance actions to ensure security and community review.

Utility

These contracts provide supporting functionality for the platform.

RewardsContract

Audited

Distributes rewards to viewers and creators based on platform engagement.

IPFSRegistry

Beta

Maintains mappings between content IDs and IPFS content identifiers.
